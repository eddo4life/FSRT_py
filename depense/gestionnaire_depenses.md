# Mini-Gestionnaire de Dépenses Personnelles

Ce programme permet d'enregistrer et de consulter des dépenses quotidiennes. Chaque dépense est stockée sous la forme :

```
(description, catégorie, montant)
```

Les catégories autorisées sont définies dans un tuple immuable.

## Fonctionnalités

### 1. Ajouter une dépense
* Saisir une description
* Choisir une catégorie autorisée
* Entrer un montant positif
* Ajouter la dépense à la liste

### 2. Afficher toutes les dépenses
* Affichage lisible : numéro, description, catégorie, montant

### 3. Quitter
* Terminer l'exécution du programme
